## 1. 서비스 관리자 (ADMIN) 플로우

> ADMIN 계정은 여러 개 존재할 수 있으며, 모두 동일한 권한을 가진다.

### 진입 경로
랜딩 → 시스템 관리 클릭 → 로그인 → 관리자 대시보드

### 화면 흐름

```
[01] 관리자 대시보드
      ├── [02] 사용자 관리           (ADMIN / PM / PROCUREMENT 계정 CRUD)
      ├── [03] 전문가 관리           (전문가 정보 등록·수정·경력 관리)
      ├── [04] 평가 관리             (평가 내역 조회·이의제기·블랙리스트)
      ├── [05] 시스템 설정           (공지사항·템플릿·약관·권한)
      └── [06] 통계·리포트
```

### 화면별 상세

#### [01] 관리자 대시보드
- 총 사용자 수 (역할별)
- 이번 달 신규 등록 전문가 수
- 진행 중인 프로젝트 수
- 이번 달 정산 완료액
- 최근 이슈 알림 (이의제기, 평가 신고 등)

#### [02] 사용자 관리
- 계정 목록 조회 (역할·상태 필터)
- 계정 생성 — ADMIN / PM / PROCUREMENT
- 계정 수정 (이메일, 역할, 상태)
- 계정 비활성화 / 재활성화
- 비밀번호 초기화

#### [03] 전문가 관리 (전문가는 로그인 불필요 — 관리자가 일괄 관리)
- 전문가 목록 조회·검색 (이름, 기술스택, 직무 필터)
- 신규 전문가 등록 (이름, 연락처, 기술스택 등 기본 정보)
- 경력 등록·수정·삭제:
  - 발주처, 프로젝트명, 직무, 기간, 기술스택
  - 수행 내용 상세
  - 증빙 서류 첨부 (계약서, 세금계산서, 검수확인서)
  - **AI 자동 등록**: 이력서 파일 업로드 시 자동 파싱
- 경력 내역서 발행 (PDF — 관리자가 출력하여 전문가에게 전달)

#### [04] 평가 관리 (블랙리스트 통합)
- 전체 평가 내역 조회
- 이의제기 접수 및 처리
- 블랙리스트 검색 (이름 / 연락처 / 이메일)
- 블랙리스트 상세: 위험도·신고이력·관련 프로젝트·관리자 메모
- 블랙리스트 상태 업데이트 (영구 정지 / 임시 제한 / 해제)

#### [05] 시스템 설정
- 공지사항 등록·수정·삭제
- 표준 계약서 템플릿 관리
- 약관·개인정보처리방침 관리
- 역할별 메뉴 접근 권한 설정

#### [06] 통계·리포트
- 플랫폼 전체 사용 현황 (MAU, DAU)
- 매칭 성공률
- 역할별 사용자 추이
- 정산 금액 추이
- 기간별 엑셀 다운로드

## 2. PM / 발주처 (PM) 플로우

### 진입 경로
랜딩 → 프로젝트 등록 클릭 → 로그인 → 프로젝트 등록

### 화면 흐름

```
[01] 프로젝트 기본 정보 — Step 1
      ↓
[02] 모집 상세 구성 — Step 2
      ↓
[03] 최종 검토 및 등록 — Step 3
      ↓
[04] 내 프로젝트 목록 / PM 대시보드
```

### 화면별 상세

#### [01] Step 1 — 프로젝트 기본 정보
- 프로젝트명
- 발주사 / 클라이언트
- 프로젝트 기간
- 예산
- 전체 기술 스택
- 프로젝트 범위 설명

#### [02] Step 2 — 모집 상세 구성
- 빠른 역할 추가: AA / TA / SA / DA / 클라우드 전문가 / 개발 PL / FE / BE 등
- 각 역할별: 필요 경력 연수, 기술 스택, 역량 요구사항, 급여 범위

#### [03] Step 3 — 최종 검토 및 등록
- 입력 정보 전체 요약 프리뷰
- 프로젝트 배경 설명 추가
- 첨부파일 업로드
- 공고 게시 확정 (승인 없이 즉시 등록)

#### [04] 내 프로젝트 목록
- 등록한 프로젝트 목록 및 상태 확인
- 프로젝트 수정 / 삭제

## 3. 구매부 (PROCUREMENT) 플로우

### 진입 경로
랜딩 → 전문가 평가 클릭 → 로그인 → 전문가 검색

### 화면 흐름

```
[01] 전문가 검색
      ↓
[02] 매칭 관리 (프로젝트별 인력 배치)
      ↓
[03] 계약 관리
      ↓
[04] 정산
      ↓
[05] 전문가 평가
```

### 화면별 상세

#### [01] 전문가 검색
- 키워드 검색 (이름, 직무, 기술 스택)
- 직무·경력연수 필터
- AI 매칭 점수 표시
- 결과 카드: 프로필, 기술 스택, 검증 이력, 원격 가능 여부

#### [02] 매칭 관리
- 프로젝트별 오픈 슬롯 / 매칭 인력 현황
- AI 기반 추천 후보자 리스트
- 전문가 프로필 상세 조회
- 매칭 제안 발송

#### [03] 계약 관리
- 계약 목록 조회 및 상태 확인
- 계약서 서명 처리
- 계약 종료 처리

#### [04] 정산
- 정산 목록 조회
- 정산 승인 / 지급 처리

#### [05] 전문가 평가 (프로젝트 완료 후)
- 5개 항목 별점 (과제 완성도·커뮤니케이션·협업·전문성·납기 준수)
- 세부 피드백 텍스트
- 재계약 의향 선택
- 결과는 전문가 프로필 신뢰도에 반영

## 4. 전체 시스템 연결 구조

```
Landing Page (공용)
├── 프로젝트 등록  → /auth/login → /app/projects/register  (PM)
├── 전문가 평가    → /auth/login → /app/evaluations        (PROCUREMENT)
└── 시스템 관리    → /auth/login → /app/admin              (ADMIN)

인증 플로우 (ADMIN / PM / PROCUREMENT만 로그인)
└── 로그인: 이메일 + 비밀번호 → JWT 발급 → 역할별 페이지 이동

앱 내 주요 라우트
├── /app                         — 역할별 리디렉션 게이트
│
├── /app/admin                   — 관리자 대시보드      (ADMIN)
├── /app/admin/users             — 사용자 관리          (ADMIN)
├── /app/admin/talents           — 전문가 관리          (ADMIN)
├── /app/admin/evaluations       — 평가 + 블랙리스트    (ADMIN)
├── /app/admin/settings          — 시스템 설정          (ADMIN)
├── /app/admin/stats             — 통계·리포트          (ADMIN)
│
├── /app/projects                — 내 프로젝트 목록     (PM)
├── /app/projects/register       — 프로젝트 등록 3단계  (PM)
│
├── /app/experts/search          — 전문가 검색          (PROCUREMENT)
├── /app/matching                — 매칭 관리            (PROCUREMENT)
├── /app/contracts               — 계약 관리            (PROCUREMENT)
├── /app/settlements             — 정산                 (PROCUREMENT)
└── /app/evaluations             — 전문가 평가          (PROCUREMENT)
```
