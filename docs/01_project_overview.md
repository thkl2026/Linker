# Linker - 외부 인력 관리 시스템 프로젝트 개요

> **슬로건**: "Reliable Stability, Intelligent Connection"  
> **비전**: 데이터와 AI를 통해 최적의 파트너십을 구축하는 지능형 외부 인력 운영 체제(Workforce OS)

---

## 1. 개발 배경 및 문제 정의

| 문제 | 설명 |
|------|------|
| 정보 파편화 | 엑셀·메일·메신저로 흩어진 이력/계약 정보 → 과거 히스토리 파악 불가 |
| 매칭 비효율 | 지인 추천·키워드 검색 의존 → 미스매칭 리스크 높음 |
| 컴플라이언스 리스크 | 개인정보보호법·노동법 대응 전자계약/증빙 체계 부재 |
| 수작업 정산 | 공수 집계·지급 오류 반복, 정산 지연 건 다수 |

---

## 2. 핵심 가치 (Core Values)

- **Reliability**: Java/Spring Boot 기반 견고한 아키텍처로 기업 핵심 데이터·계약 보호
- **Intelligence**: AI가 인력 분석·매칭·프로젝트 리스크 예측, AI 챗으로 자연어 업무 처리
- **Efficiency**: 파편화된 외부 인력 관리 프로세스 단일 플랫폼으로 통합
- **Accessibility**: 웹·모바일 크로스플랫폼 지원으로 언제 어디서나 접근 가능

---

## 3. 정량적 목표 (KPI)

| 지표 | 목표 |
|------|------|
| 소싱 → 계약 리드타임 | 50% 단축 |
| AI 추천 인력 투입 확정률 | 80% 이상 |
| 정산 지연 건수 | 0건 |
| 리크루팅 시간 | 70% 이상 단축 |
| 모바일 앱 주요 업무 처리율 | 가용 상태 변경·보고 제출·알림 확인 100% 모바일 대응 |
| AI 챗 태스크 해결률 | 챗봇 단독 해결 70% 이상 (인력 검색·계약 조회·정산 문의) |

---

## 4. 액터 정의

| 액터 | 정의 | 주요 역할 |
|------|------|-----------|
| 외부 인력 (Talent) | 프리랜서 / 파트너사 직원 | 프로필 관리, 가용 상태 업데이트, 프로젝트 수행, 정산 요청 |
| 프로젝트 매니저 (PM) | 현업 실무 부서 | 프로젝트 등록, 인력 검색·선발, 실무 진척 관리 |
| 구매부 (Procurement) | 계약·정산 전담 부서 | 파트너사 등록 승인, 단가 협상, 계약 확정, 정산 최종 승인 |
| 운영 관리자 (Admin) | 시스템·프로세스 운영팀 | 권한 관리, 표준 양식 관리, 전사 통계 모니터링 |
| AI 엔진 (System) | 지능형 보조 시스템 | 이력서 분석, 스코어링, 인력 추천, 리스크 감지 |

---

## 5. 지원 플랫폼

| 플랫폼 | 기술 | 주요 대상 |
|--------|------|-----------|
| **웹 (PC)** | React 18 + Vite (SPA) | PM·구매부·Admin — 풀 기능 |
| **모바일 앱 (iOS/Android)** | React Native (Expo) | 외부 인력 — 가용 상태·보고·알림·정산 확인 중심 |
| **모바일 웹 (PWA)** | React + Vite PWA Plugin | 앱 미설치 사용자 대상 경량 접근 |

> 가입 시 **NICE/KCB 실명인증** + **2FA(TOTP 또는 SMS OTP)** 필수 완료 후 서비스 이용 가능

---

## 6. 시스템 모듈 구성

```
Linker
├── Module 1: 인력 및 역량 관리 (Talent Management)
├── Module 2: 사업 기회 및 매칭 (Opportunity & Matching)
├── Module 3: 계약 및 구매 관리 (Contract & Procurement)
├── Module 4: 수행 및 피드백 검증 (Execution & Feedback Validation)
├── Module 5: 정산 및 통계 (Settlement & Analytics)
└── Module 6: AI 챗 어시스턴트 (AI Chat Assistant)  ← 신규
```
