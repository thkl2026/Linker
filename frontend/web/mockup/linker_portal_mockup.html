Linker User Flow Guide

사용자 역할에 따른 최적화된 동선을 확인하고 목업을 탐색해보세요.

🧑‍💻

전문가 (Expert)

자신의 커리어를 인증하고 리포트를 발행하여 신뢰를 얻습니다.

Main Flow

01

커리어 대시보드

전체 경력 요약, 역량 정보 확인 및 관리

→
02

경력 등록 (상세)

참여 프로젝트의 역할, 기술, 기여도 등록 및 인증 요청

→
03

경력 내역서 발행

인증된 정보를 바탕으로 프리미엄 리포트 다운로드

→
🏢

발주처 / PM (Client)

프로젝트를 정의하고 최적의 전문가 팀을 빌딩합니다.

Main Flow

01

프로젝트 기본 정보 (Step 1)

명칭, 발주사, 전체 기술 스택 및 범위 정의

→
02

모집 상세 구성 (Step 2)

AA, PL 등 다양한 전문 인력의 상세 요건 구성

→
03

최종 검토 및 등록 (Step 3)

배경 설명 추가 및 프로젝트 공고 게시

→
